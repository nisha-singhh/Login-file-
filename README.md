🔐 Secure User Authentication System

A modern and responsive authentication system built with Streamlit and SQLite. The application provides secure user registration and login functionality with an attractive glassmorphism user interface and persistent user data storage.

🚀 Live Demo

Application Link: https://ezvdmrxkgcespqfr9ddmh2.streamlit.app/

📌 Project Overview

The Secure User Authentication System allows users to create accounts and securely log in using their credentials. User information is stored in an SQLite database, enabling persistent authentication across sessions.

The project demonstrates the implementation of:

User Registration
User Login Authentication
SQLite Database Integration
Session State Management
Responsive UI Design
Glassmorphism Styling
Custom CSS in Streamlit
